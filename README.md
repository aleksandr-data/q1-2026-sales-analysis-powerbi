# Q1 2026 Sales Analysis | Excel, Power Query, DAX, Power BI

## О проекте
Учебный end-to-end проект по анализу продаж за Q1 2026 на synthetic dataset.

Проект охватывает полный базовый цикл:
- работа с исходными данными в Excel;
- подготовка и очистка в Power Query;
- построение модели данных;
- создание мер на DAX;
- сборка overview-dashboard в Power BI.

## Цель проекта
Собрать портфельный проект, который показывает базовый рабочий цикл аналитика данных:
от исходного датасета до аналитической модели и dashboard.
